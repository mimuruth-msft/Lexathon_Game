To run the program:

1. Make sure everything is in the SAME FOLDER.
2. Open MARS from within the folder.
3. Open LexathonGame.asm
4. Navigate to the Settings menu in MARS.
5. Make sure "Assemble all files in directory" is checked.
6. Make sure "Initialize Program Counter to global 'main' if defined" is checked.
7. Assemble the program (F3).
8. Run the program (F5).

Troubleshooting:
If you get the error: "Symbol...not found in symbol table.", make sure you have done steps 4 and 5.
If you get the error: "Error in: Invalid program counter value: 0", make sure you have done steps 4 and 6.
